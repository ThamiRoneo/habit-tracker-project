# Habit Tracker

A beginner-friendly Habit Tracker web application built with HTML, CSS, and Vanilla JavaScript. This project allows users to create daily habits, track completion streaks, monitor progress, and view real-time summary statistics—all directly in the browser without any frameworks, backend services, or databases.

## Features

* Add new habits with a name, category, and weekly target.
* Client-side form validation with inline error messages.
* Mark habits as completed for the day.
* Automatic streak tracking and updates.
* Delete habits from the tracker.
* Real-time summary dashboard displaying:

  * Total habits
  * Habits completed today
  * Overall completion percentage
* Dynamic rendering using JavaScript and DOM manipulation.
* Single source of truth architecture using an array of habit objects.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
